## ADDED Requirements

### Requirement: Public site refresh
The system SHALL present a refreshed public UI for the home page and public leaderboard with improved layout, typography, and responsive design.

#### Scenario: Home page view
- **WHEN** a user visits the home page
- **THEN** the page presents clear calls to action for admin login and the public leaderboard

#### Scenario: Public leaderboard view
- **WHEN** a user views the public leaderboard
- **THEN** the layout remains readable and usable on both desktop and mobile

### Requirement: Admin portal refresh
The system SHALL present a refreshed admin UI with consistent layout, navigation, and data views.

#### Scenario: Admin page view
- **WHEN** an admin views a portal page
- **THEN** the UI aligns with the shared visual system and preserves core workflows

### Requirement: Admin overview page
The system SHALL provide an admin overview page that surfaces system summary metrics.

#### Scenario: Admin overview
- **WHEN** an authenticated admin opens the overview page
- **THEN** the UI displays system summary data from the backend
